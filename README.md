# E-Commerce Website - Backend

## Overview

The E-Commerce Website Backend is a RESTful API developed using Spring Boot. It manages core e-commerce functionalities such as user authentication, product management, category management, shopping cart, order processing, and database operations. The backend provides secure APIs that interact with the frontend application.

## Features

* User Authentication & Authorization
* Product Management
* Category Management
* Shopping Cart Management
* Order Management
* Customer Management
* RESTful API Development
* Input Validation
* Exception Handling
* Database Integration

## Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Maven
* Lombok

## Getting Started

1. Clone the repository.
2. Configure the MySQL database in `application.properties`.
3. Build the project using Maven.
4. Run the Spring Boot application.
5. Access the REST APIs from the frontend.
